Informationen

Diese Applikation steht kostenlos zur Verfügung. Die Applikation dient dazu SEPA Überweisungsdaten als GiroCode (EPC QR Code) anzuzeigen. Die meisten Mobile Banking Apps von europäischen Banken sind in der Lage den Code zu Scannen. Bei den deutschen Volksbanken Reifeisenbanken findet man die Funktion z.B. unter "Überweiseung, Fotoüberweisung".

Mit dem Scannen eines GiroCodes wird lediglich das Überweisungsformular automatisch und ohne Tippfehler ausgefüllt. Die Überweisung wird erst ausgeführt, wenn das ausgefüllte Formular auch tatsächlich abegesendet wird und die Überweisung mit einer TAN bestätigt wird.

Alle eingegeben Daten werden offline auf dem Gerät und in dem Webbrowser gespeichert, in dem sie eingegeben wurden.

Die Daten werden auch in Textform angezeigt. Sollte also jemand nicht in der Lage sein, einen QR-Code zu scannen, können die Daten manuell eingegeben werden.

Installation

Diese Anwendung ist eine sogenannte "Progressive Web App". Sie kann wie eine reguläre App aus dem App-Store benutzt werden.

Für die Installation auf iOS-Geräten (iPhone, iPad und iPod Touch) öffnet man diese Anwendung als Web-Seite in Safari.

  • Dort tippt man auf das "Teilen"-Symbol (mittig, unten).
  • In der Teilen-Funktion wählt man "Zum Home-Bildschirm".
  • Im nachfolgenden Bestätigungsdialog kann man den Namen der App ändern und anschließend wählt man "Hinzufügen" (rechts, oben).
  • Die Installation ist damit abgeschlossen.

Auf einem Android-Gerät mit Chrome-Browser öffnet man die Anwendung als Web-Seite in Chrome.

  • Dort öffnet man das Menü (recht, oben, drei Punkte untereinander).
  • Im Menü wählt man die Option "Zum Startbildschirm hinzufügen".
  • Im nachfolgenden Bestätigungsdialog tippt man auf "Hinzufügen".
  • Die Installation ist damit abgeschlossen.

Die Installation funktioiert auf anderen Geräten ähnlich.
